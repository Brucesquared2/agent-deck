# 40-knowledge — Curated Knowledge

Reusable facts and patterns promoted from session logs.

## Structure
```
40-knowledge/
├── patterns/   ← code/architecture patterns worth reusing
├── tools/      ← tool usage notes and gotchas
└── research/   ← PI research findings (promoted from signals)
```

## Rules
- Items are promoted here by the Implementer or PI after a session.
- Every file must have at least one tag from `TAG_INDEX.md`.
- Knowledge is mutable — update in place when a fact changes.
- Archive outdated knowledge to `90-archive/` rather than deleting.
