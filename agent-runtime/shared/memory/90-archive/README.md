# Archive

> Entries moved here are no longer actively maintained but are preserved for audit purposes.
> Do not edit archived entries; add a new entry in the appropriate active layer instead.

## Structure

Files are stored as `<original-layer>/<original-filename>`, e.g.:

```
90-archive/20-projects/my-old-project/project-pack.md
90-archive/10-ops/2024-06-01-deprecated-runbook.md
```
